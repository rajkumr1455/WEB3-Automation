# Signature Generator

Convert security findings into monitoring signatures and detection rules for various platforms.

## Features

- **YARA Rules**: Bytecode pattern matching
- **Sigma Rules**: Transaction monitoring  
- **Suricata Rules**: Network-level detection
- **Custom Rules**: Platform-agnostic JSON format

## Quick Start

Service runs on port 8012. Generate signatures from security findings using multiple output formats.
