# Streaming Indexer

Real-time blockchain event indexing with WebSocket streaming support.

## Features

- **Real-Time Indexing**: Live event streaming from blockchains
- **WebSocket Support**: Subscribe to event streams
- **Multi-Chain**: Support for 5+ blockchains
- **Historical Backfill**: Index past events
- **Query API**: Flexible event querying

## Quick Start

Service runs on port 8014 with WebSocket support at `/ws`.
