# Remediator Service

Automated vulnerability remediation with GitHub PR creation.

## Features

- **Automated Fix Generation**: Generates fixes for common vulnerabilities
- **GitHub Integration**: Creates pull requests automatically
- **Fix Verification**: Validates fixes before submission
- **Multiple Vulnerability Types**: Reentrancy, access control, overflow, etc.

## Quick Start

Service runs on port 8013. Requires GITHUB_TOKEN environment variable for PR creation.
