# Quick Reference: RPC Configuration for .env

## Copy These Lines to Your `.env` File

Add these RPC endpoint configurations (replace YOUR_ALCHEMY_KEY and YOUR_INFURA_KEY with your actual API keys):

```bash
# ============================================
# Blockchain RPC Endpoints (with Failover)
# ============================================
# Primary RPC endpoints
ETHEREUM_RPC_URL=https://eth-mainnet.g.alchemy.com/v2/YOUR_ALCHEMY_KEY
BSC_RPC_URL=https://bsc-dataseed.binance.org/
POLYGON_RPC_URL=https://polygon-rpc.com/
ARBITRUM_RPC_URL=https://arb1.arbitrum.io/rpc
OPTIMISM_RPC_URL=https://mainnet.optimism.io
BASE_RPC_URL=https://mainnet.base.org

# Backup RPC endpoints (for automatic failover)
ETHEREUM_RPC_URL_BACKUP=https://mainnet.infura.io/v3/YOUR_INFURA_KEY
BSC_RPC_URL_BACKUP=https://rpc.ankr.com/bsc
POLYGON_RPC_URL_BACKUP=https://polygon-mainnet.public.blastapi.io
ARBITRUM_RPC_URL_BACKUP=https://rpc.ankr.com/arbitrum
OPTIMISM_RPC_URL_BACKUP=https://rpc.ankr.com/optimism
BASE_RPC_URL_BACKUP=https://base.publicnode.com

# Legacy (backward compatibility)
ETH_RPC_URL=https://eth-mainnet.g.alchemy.com/v2/YOUR_ALCHEMY_KEY
```

## Free RPC Endpoints (No API Key Needed)

If you don't have Alchemy/Infura keys yet, you can use these public RPCs:

```bash
# Free public RPCs (rate-limited but functional)
ETHEREUM_RPC_URL=https://eth.llamarpc.com
ETHEREUM_RPC_URL_BACKUP=https://ethereum.publicnode.com

BSC_RPC_URL=https://bsc-dataseed.binance.org/
BSC_RPC_URL_BACKUP=https://rpc.ankr.com/bsc

POLYGON_RPC_URL=https://polygon-rpc.com/
POLYGON_RPC_URL_BACKUP=https://polygon-mainnet.public.blastapi.io
```

## Get Free API Keys

- **Alchemy**: https://dashboard.alchemy.com/ (300M compute units/month free)
- **Infura**: https://infura.io/register (100k requests/day free)

## Where to Add in `.env`

Insert after the `# Service URLs` section and before `# Blockchain API Keys`.

Your `.env` file structure should look like:
```
...
ADDRESS_SCANNER_URL=http://address-scanner:8008

# [INSERT RPC URLS HERE]

# Blockchain API Keys
ETHERSCAN_API_KEY=...
```
